# 🧦 SOCKS5 Proxy Installer — Dante

Автоматическая установка и настройка SOCKS5-прокси с аутентификацией на базе **Dante (danted)** для Ubuntu / Debian.

---

## ✨ Возможности

- 🔍 Проверяет версию `dante-server` в репозитории (требуется ≥ 1.4.x)
- 🌐 Автоматически определяет сетевой интерфейс и внешний IP
- 🔐 Генерирует случайные или принимает ручные логин, пароль и порт
- ✅ Подтверждение пароля при ручном вводе
- 📋 Три уровня логирования: только ошибки / подключения / подробный
- 🗂 Опциональная запись логов в файл с автоматической ротацией (`logrotate`)
- 🔥 Открывает порт в UFW (если активен)
- 💾 Создаёт бэкап предыдущего конфига при переустановке
- 🩺 Проверяет успешность запуска службы с выводом диагностики при ошибке

---

## 📋 Требования

| Параметр | Значение |
|---|---|
| ОС | Ubuntu 20.04 / 22.04 / 24.04, Debian 11 / 12 |
| Права | `root` / `sudo` |
| Пакет | `dante-server >= 1.4.x` (из стандартных репозиториев) |
| Прочее | `curl`, `ufw` (опционально) |

## 📥 Установка

```bash
wget -O setup_socks5.sh https://raw.githubusercontent.com/omichs/dante-auto/main/setup_socks5.sh \
  && chmod +x setup_socks5.sh \
  && sudo ./setup_socks5.sh
```

## 🚀 Использование

После установки скрипт выведет данные для подключения:

```

=============================================================
  IP-адрес : 123.123.123.123
  Порт     : 41782
  Логин    : ab3xK9mZ
  Пароль   : Qw8!vR2@nLpX#4Yj
-------------------------------------------------------------
  Строки для антидетект-браузеров:
  123.123.123.123:41782:ab3xK9mZ:Qw8!vR2@nLpX#4Yj
  socks5://ab3xK9mZ:Qw8!vR2@nLpX#4Yj@123.123.123.123:41782

  curl -x socks5://ab3xK9mZ:Qw8!vR2@nLpX#4Yj@123.123.123.123:41782 https://ifconfig.me

```

### Полезные команды после установки

```bash
# Статус службы
systemctl status danted

# Просмотр логов в реальном времени
tail -f /var/log/danted.log
# или через journald (если выбран syslog)
journalctl -u danted -f

# Редактирование конфигурации
nano /etc/danted.conf
systemctl restart danted

# Перезапуск
systemctl restart danted
```

## ❌ Удаление

```bash
wget -O uninstall_socks5.sh https://raw.githubusercontent.com/omichs/dante-auto/main/uninstall_socks5.sh \
  && chmod +x uninstall_socks5.sh \
  && sudo ./uninstall_socks5.sh
```

Скрипт удаления автоматически:
- Останавливает и удаляет службу `danted`
- Удаляет пакет `dante-server` и конфигурационные файлы
- Находит и (с подтверждением) удаляет системных пользователей прокси
- Удаляет правила UFW для использовавшегося порта
- Предлагает перезагрузку сервера

---

## ⚙️ Что настраивается во время установки

| Шаг | Описание |
|---|---|
| Логин | Ручной ввод или генерация (буква + 6 символов) |
| Пароль | Ручной ввод (с подтверждением, мин. 8 символов) или генерация (16 символов) |
| Порт | Ручной ввод (1024–65535) или случайный свободный из диапазона 10000–64511 |
| Логирование | `error` / `error connect disconnect` / `error connect disconnect data` |
| Вывод логов | Файл `/var/log/danted.log` (с ротацией) или `syslog` |

---

## 🔒 Безопасность

- Пользователь создаётся с shell `/usr/sbin/nologin` — вход в систему невозможен
- Пароль устанавливается через `chpasswd` (не через `echo | passwd`)
- `internal:` привязан к конкретному сетевому интерфейсу, а не к `0.0.0.0`
- Аутентификация обязательна для всех подключений (`socksmethod: username`)
